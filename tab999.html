Credits

This is a collection of old smiles mostly used in persian forums.

Special thanks to PersianTools members for providing and collecting smiles. Some of smiles collected from AsefSoft QuickSmiles, Worldofugly.de, BlackMice and other smiles and forum websites.

This collection is free to use. If you have any complaints, please contact us here: https://github.com/retrosmiles/retrosmiles/issues/


How To Use

Click on the smiley to copy BBCode to clipboard then go to forum's editor and Paste code there.

Or install and use Extensions.

Note: your forum should support BBCode.


Addition Request

If you want to add more smiles, fork us, add smiles and edit html files (you can use droplet.html to generate html codes quickly), and then send your pull request here: https://github.com/retrosmiles/retrosmiles/pulls or send your request to this topic

For extension support visit this topic.
